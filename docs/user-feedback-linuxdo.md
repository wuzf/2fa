## 一、功能需求（Feature Requests）

### 高优先级

#### 1. WebDAV / S3 加密备份到云盘
- **提出者**: zhubaiwan-oozzxx (#51, #54), 952777 (#114), chuanqibuding (#115)
- **描述**: 多人反复提出，认为数据只存在 CF KV 不够安全，希望能定时备份到 WebDAV 或 S3 云盘，支持加密
- **原始反馈**:
  - zhubaiwan-oozzxx: "能否 webdev 定时备份？"
  - zhubaiwan-oozzxx: "所有数据都保存在 cf 也不算很安全，希望佬友能增加一个 webdev 备份"
  - 952777: "不懂就问，这种能做成连接webdav自动把加密过的数据备份到云盘上吗？"
  - chuanqibuding: "最好能够webdav或S3加密备份"
- **参考**: ruchu8/2fa 已实现 GitHub Actions 版本的 S3/WebDAV/GitHub 多后端备份同步，但质量不高
- **状态**: 待开发

#### 2. ~~修复模态框误关闭 Bug~~ ✅ 已修复
- **提出者**: chuanqibuding (#101)
- **描述**: 手动添加密钥时鼠标不小心点到窗口外面，添加窗口就消失了，填写的数据丢失
- **原始反馈**: "点击手动添加 鼠标不小心点到外面 手动添加窗口就消失了。这个可以修复下吗？"
- **状态**: ✅ 已修复（`320b140` — 禁用点击模态框外部关闭）

### 中优先级

#### 3. ~~PC端截图/粘贴识别二维码~~ ✅ 已实现
- **提出者**: chuanqibuding (#115, #116)
- **描述**: 方便电脑用户，截图或复制二维码图片后直接粘贴识别
- **原始反馈**: "建议新增PC端截图识别二维码。方便电脑用户。"
- **参考**: ruchu8/2fa 已有实现（handleGlobalPaste, triggerPasteQR, decodeQRCodeFromBlob），但代码质量不高，建议自行重新实现
- **状态**: ✅ 已实现 — 扫码弹窗和二维码解析工具均支持剪贴板粘贴 + 拖拽图片识别二维码

#### 4. 登录支持 2FA 验证 / 防爆破
- **提出者**: lewss (#82)
- **描述**: 担心密码被爆破，建议登录本身也支持 2FA 验证
- **原始反馈**: "登录支持2fa吗？密码还是怕爆破。"
- **状态**: 待评估（目前已有 rate limiting 5次/分钟）

#### 5. 分类和收藏功能（常用密钥置顶）
- **提出者**: kitwing (#53)
- **描述**: 密钥多时需要分类管理，常用的可以置顶/收藏
- **原始反馈**: "佬，希望增加个分类和收藏（把常用的置顶）"
- **回复**: 你(#122): "项目一开始只有我在用，因为我管理150+个密钥...分类和收藏，包括置顶一开始就没考虑过，后面我再认真想一下"
- **状态**: 待开发

#### 6. 项目更新流程简化
- **提出者**: kitwing (#53)
- **描述**: 用户不太懂 CF 上怎么更新版本
- **原始反馈**: "如果项目更新了，CF上怎么更新，不太懂CF"
- **回复**: 你(#122): "把workers项目删除即可(KV千万不要动)，然后重新一键部署"
- **状态**: 可考虑提供一键更新方案或文档
